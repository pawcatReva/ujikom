# Herby

## Konsep Website
Website yang saya buat ini merupakan sebuah website tempat membeli tumbuhan atau rempah-rempah.

## Fitur Yang tersedia
- Topbar
   - Email/mail
   - Nomor Kontak
   - Tempat lokasi
   - Facebook
   - Instagram
   - Youtube
- Halaman Utama
   - Halaman Beranda (Home)
   - Tentang (about)
   - Belanja (Shopping)
   - Keranjang
   - Akun (account)
  
- Multi user
   - User/pengguna Herby
      - Register (membuat akun sebaga pembeli)
      - Login setelah register
      - Logout (sebagai pembeli)
      - Membeli produk
      - Membuat pesanan
      - Mengelola akun
    
  - Admin
     - Login (sebagai admin)
     - Pengguna yang dapat dikelola
     - Membuat pembaruan stok barang
     - Pesanan yang dapat dikelola
